# Edge-MDS Misbehavior Detection

This repository contains the implementation of the proposed Edge-MDS approach for misbehavior detection in vehicular networks, along with a trajectory-compatible baseline derived from GOLIATH [Maffiola et al., 2021].

## Dataset
We use the NGSIM vehicle trajectory dataset.
